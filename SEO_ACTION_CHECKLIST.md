## Emergency Troubleshooting

### Site Not Showing in Google?
1. Check Google Search Console for errors
2. Verify sitemap is submitted
3. Request indexing manually
4. Check robots.txt isn't blocking
5. Wait 2-4 weeks for initial indexing

### Rankings Dropped?
1. Check for Google algorithm updates
2. Review Search Console for penalties
3. Check if site is mobile-friendly
4. Verify all pages load properly
5. Review recent content changes

### No Traffic from SEO?
1. Verify analytics is working
2. Check keyword difficulty (may be too high)
3. Review content quality
4. Build more backlinks
5. Focus on easier long-tail keywords
